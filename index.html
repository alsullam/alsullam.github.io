About

Why We Build From Scratch

AL Sullam — "The Stair" in Arabic — is a systems programming organization with a single guiding principle: understand everything below you.

We live in an era of spectacular abstractions. Frameworks, runtimes, managed languages — each a gift that hides the machine underneath. We're grateful for them. We also refuse to be mystified by them.

Every project starts at zero: no borrowed libraries for core logic, no generated parsers, no magic hash functions imported from somewhere else. We write the SHA-1. We write the TCP loop. We write the lexer character by character.

Not because it's faster. Because it teaches. درجة بدرجة — step by step.

  • 01

    No Black BoxesEvery dependency must be understood. If we use it, we could write it.

  • 02

    Document EverythingCode without explanation is archaeology. Every project ships with deep-dive docs explaining the why, not just the how.

  • 03

    The Spec Is the AuthorityRFC, BEP, ISO — read the actual specification, implement faithfully, test against official vectors.

  • 04

    Low-Level FirstStart in C, where nothing is hidden. Understand memory, addresses, syscalls. Higher levels become obvious after.

  • 05

    Correctness Over PerformanceMake it right before making it fast. Working SHA-1 beats broken SIMD.
